This project is a desktop-based Learning Management System (LMS) developed as a semester project using C++ and the Qt Framework. The application is designed to demonstrate core software engineering and object-oriented programming concepts through a role-based academic management system.

The LMS provides separate access and functionality for Admin, Faculty, and Students, allowing structured interaction between different users within a simulated university environment. The system focuses on clean design, modular architecture, and GUI-based interaction.

Key Features

Role-based access control (Admin, Faculty, Student)

Dedicated dashboards for each user role

Student and faculty record management

Assignment and academic data handling

File-based data storage for persistence

User-friendly graphical interface built with Qt Widgets

Technologies and Concepts Used

C++ (Object-Oriented Programming)

Qt Framework (Qt Creator)

GUI development using .ui files

File handling for data storage

Modular class-based system design

Project Objective

The main objective of this project is to apply theoretical knowledge of:

Object-Oriented Programming principles

GUI-based application development

Role-based system design

Semester-level software architecture using C++ and Qt

How to Run

Open the project in Qt Creator

Select a compatible Qt version (Qt 6 recommended)

Configure a supported compiler (MinGW or equivalent)

Build and run the project
